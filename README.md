# Computer Vision Powered Image Search Application

This is a portfolio project demonstrating an end-to-end MLOps pipeline. It features a custom Object Detection search engine powered by Ultralytics YOLOv11, PyTorch, and a Streamlit frontend, deployed securely via Docker containers.
